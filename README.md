# Apple and Basket — Frontend


## Tecnologías
- Svelte
- Vite
- JavaScript

---

## Requisitos
- Node.js 18+ (recomendado)
- npm (o pnpm/yarn)

---

## Configuración (URL del Backend)

El frontend necesita apuntar a la URL del backend desplegado (Railway) o local en el archivo "estado.js"

Asegúrate de que el frontend use la URL del backend:

- https://proyectofinal-backend-production-91d7.up.railway.app


## Ejecutar en local:
Instalar dependencias 
* npm install

Levantar en servidor local:
* npm run dev
Abre en el navegador: http://localhost:5173

Compilar:
* npm run build
